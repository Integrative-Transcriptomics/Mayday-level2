This tool generates statistics about your loaded SAM/BAM files with the
picard tool.
However, you must still specify some details about your files:

Strand specificity of your library:
Use FIRST_READ_TRANSCRIPTION_STRAND if the unpaired reads or the first
read of a pair are expected to be on the transcription strand.

Refflat:
Refflat is a small annotation file that you can download from
http://hgdownload.cse.ucsc.edu/downloads.html .
Download the refFlat.txt.gz from under the
Annotation database section of your corresponding organism and unzip it.

Reference (optional):
Some metrics, such as the mismatche rate, require a the fasta reference
file. That file can be omitted, because it might not be available or
is too large. If you do not specify a reference, the dependent metrics are zero.
